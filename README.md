## 🎯 O que me move

Construo plataformas, não features. Cada projeto tem arquitetura pensada para escalar, observabilidade desde o dia 1, e ciclo de vida bem definido — não só um MVP que "funciona". Gosto de problemas difíceis: alocação de recursos com restrições, fluxos assíncronos complexos, integração de IA em pipelines reais.

**Princípios de trabalho:**
- **Monorepo-first**: Turborepo + Bun, pacotes compartilhados, build incremental
- **Type-safe ponta a ponta**: oRPC/tRPC, Zod em toda fronteira, nunca um `any`
- **Async by default**: BullMQ para o que pode ser assíncrono, SSE para updates em tempo real
- **IA como ferramenta**: agentes com streaming, tool use, LLMs para filtrar/classificar, não para substituir lógica
